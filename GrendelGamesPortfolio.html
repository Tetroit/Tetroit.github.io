Itch

A small horror narrative that takes place in the Northern-Mill of Twickel, Twente. Made for Tweeduuster collection. I was responsible for player controller, animation programming and partially graphics (custom postprocessing in particular). During this project I used extra Unity packages such as DOTween and Unity Timeline to achieve smooth and robust animations.

  • player FSM with multiple controller types (physics based, ladder), support for camera shake and animation transitions
  • postprocessing shaders to enhance the game feel: edge distortion, eye clipping
  • Quick time event and animation system compatible with Unity Timelines

Procedural village

March 2024 - May 2024

Solo project, made with Unity v6 URP in 2 months >>>

GitHub

A fully procedurally generated village with houses roads and details made in Unity. The program runs with a single editor button click, has a lot of features and customization, such as:

  • randomly oriented house areas
  • automatic terrain alphamap road paint
  • procedurally generated houses with voxel architecture
  • terrain height accommodation
  • detail placement
  • spline tool for easy custom wooden bridges

This was a very interesting and technical project, and I will gladly disscuss it as a whole!

Ghost Box

March 2024

Group project, multidisciplinary 7-member team, made with Unity v5 URP in 3 weeks >>>

GitHub

An Android puzzle game made in Unity that uses mobile sensors: light sensor, gyroscope, and accelerometer. Players have to solve puzzles on the box by tapping, dragging or tilting the phone. If they make real environment dark, the box will also switch to dark mode revealing the clues! If the player stay too long in the dark the ghost consumes them and they lose.
During the project I created night mode shaders using Shader Graph and custom HLSL functions, a set of custom postprocessing effects for better ambience, reading sensor input from the device and light sensor calibration sequence. Here are linked my calibration, game manager, sensor input codes as the most outstanding parts of my work for this project.
