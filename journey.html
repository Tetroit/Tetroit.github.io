Unnamed Jellyfish Game

A solo project, made in C# using GXP Engine (Saxion University's 2D engine). A simple arcade game with coins and highscore. You control a jellyfish and has to maneuver through the scrolling down map and avoid spikes and fireballs. The map is infinite and randomly generated (using perlin noise) with 2 generation types and different structures.

Cheesekeeper

A group project. An arcade game made in C# using GXP Engine (Saxion University's 2D engine) fully operated by a custom controller based on Arduino Nano 33 iot. Draw shapes in the air using controller and defeat coming upcoming enemies with magic. The game has advanced scoring system supported by visuals, has ultimate abilities and a leaderboard

Neodymium

A group project made in C# using GXP Engine (Saxion University's 2D engine). A 3D physics based puzzle game. Control a crane to grab and drop objects in order to create a path for a robot. Engine was remade to support 3D. The game has a path following robot, fully controllable crane and multiple physics powered objects.
